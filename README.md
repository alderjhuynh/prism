# Prism

A keyboard-played crystal synth that runs entirely in a single HTML file, built on [Tone.js](https://tonejs.github.io/).

## Overview

Prism is a browser-based synthesizer and mini music studio. Play notes live on a glowing crystal, compose melodies on a piano roll, build beats on a drum machine, and arrange everything together. no install, no build step.

## Features

- **Live** — Play notes by clicking the crystal shards or using the keyboard. Shift octaves up/down on the fly.
- **Compose** — Write melodies on a piano roll, with a fullscreen view for more detailed editing.
- **Drums** — A per-voice step sequencer / piano roll for programming beats, with switchable voices.
- **Arrange** — Combine your composed melody and drum pattern into a single arrangement.

## Controls

- **A W S E D F T G Y H U J K O L P ;** — play notes
- **Z / X** — shift octave down / up
- Click or hold a crystal shard with the mouse to preview a pitch class
- Click **enable audio** once at the start (required by browsers before sound can play)

## Tech

- Single self-contained HTML file (markup, styles, and script all inline)
- Audio powered by [Tone.js](https://tonejs.github.io/)
